# Mission-Plan-Programming-For-Drone-Application
Mission plan programming using Pixhawk 2.4.8 flight controller, capable of autonomous waypoint navigation and flight path execution.

## Objective 
Mission plan programming using Pixhawk 2.4.8 flight controller, capable of autonomous waypoint navigation and flight path execution.

## Hardware components 
1. Hexacopter Frame
2. Pixhawk 2.4.8
3. 3DR Telemetry Radio
4. ReadytoSky GPS Module
5. Safety Switch
6. Buzzer
7. Lithium Polymer Batteries (24000maH)
8. LiPo battery charger
9. Powwer module
10. Transmitter FS-i6S
11. Receiver FS-iA 10B
12. Electonic speed controllers
13. Brushless DC motors
14. Propellers

## Technologies used
1. Mission planner software
2. Drone APIs
3. Waypoint configuration
4. Communication protocols

## Working principle
- The electronics are connected to the Pixhawk 2.4.8 flight controller.
- Connection is established between flight controller and mission planner software.
- Mission planner software is configured.
- Parameters are set to prevent runway UAVs and fail-safe modes are enabled.
- Waypoints are predefined for an autonomous flight.

## Mission planner Workflow
1. Parametrs like pitch, yaw, roll are configured in mission planner software.
2. Define waypoint coordinates in the mission planner software.
3. Upload mission commandes to the flight controller.
4. Waypoints navigation is implemented for autonommous flight.
5. Monitor drone path and mission completion.
